# Deployment and CI/CD

- Run CI with automated tests on every commit.
- Use CD for staging and deploy to staging before production.
- Keep environment-specific configuration secure (e.g. Azure Key Vault).
- Use GitHub Actions or Azure DevOps.

> Source: [Justice Digital software engineering standards](https://technical-guidance.service.justice.gov.uk/documentation/standards/justice-digital-software-engineering-standards.html)
