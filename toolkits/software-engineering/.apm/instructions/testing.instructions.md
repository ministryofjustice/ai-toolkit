# Testing Standards

- Code reviews are mandatory before merge.
- Write integration tests for major workflows.
- Use automated frameworks (e.g. JUnit, PyTest).
- Confirm unit test coverage for critical modules with the technical lead.
- Tests and assertions must be meaningful.

> Source: [Justice Digital software engineering standards](https://technical-guidance.service.justice.gov.uk/documentation/standards/justice-digital-software-engineering-standards.html)
