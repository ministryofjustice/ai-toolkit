# JavaScript / TypeScript Frontend

- Enforce strict typing with TypeScript.
- Use ES6+.
- Implement error handling; avoid silent failures.
- Avoid global variables; use modules or closures.
- Use ESLint.
- Prefer React, Angular, or Vue as approved; use Redux or Zustand for state management.

> Source: [Justice Digital front end engineering standards](https://technical-guidance.service.justice.gov.uk/documentation/standards/justice-digital-front-end-engineering-standards.html)
